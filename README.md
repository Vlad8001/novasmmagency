# Nova Social Media Agency — сайт

Статичний сайт (HTML + CSS + JS) з повноцінними анімаціями та адаптивом
для мобільних та планшетів.

## Що всередині

```
nova-site/
├── index.html      ← вся розмітка, SVG-символи вбудовані
├── styles.css      ← 1200+ рядків: тема, анімації, 5 breakpoint'ів
├── script.js       ← лічильники чисел, donut, лінія процесу, форма
└── assets/         ← папка порожня — все SVG інлайн
```

Фото засновниці підтягується з Framer CDN автоматично. Якщо будете міняти
хостинг і CDN стане недоступним — просто збережіть фото як `assets/founder.jpeg`
і в `index.html` замініть URL в тегу `<img>` секції `#trust`.

## Анімації, які тепер працюють

**Постійні:**
- Сфера у hero-картці повільно обертається (22 сек цикл)
- Іконка-зірка у міні-картці "Стратегія → контент" крутиться (30 сек)
- Сфера-декор під карткою "Зйомка" також обертається
- 4 декоративні зірки у секції прайсу гойдаються вгору-вниз з поворотом
- Зірка в секції відгуків гойдається

**На появу при скролі:**
- Заголовки і leadи плавно fade-in знизу
- Картки в рядах з'являються послідовно (stagger по 80ms)
- Числа рахуються від 0 до кінцевого (5+, 54.6K, 390.5K, 20.4K, 277+, 2.4 млн)
- Donut chart малюється (2.4 млн з дельтою +18.6%)
- Прогрес-бари під числами наповнюються шириною
- Пунктирна хвиляста лінія у процесі малюється зліва направо

**На завантаження:**
- Hero-елементи послідовно виїжджають знизу (тег → заголовок → підзаголовок → кнопки → картка)

**Interactive:**
- Кнопки: підйом на hover, стрілка ↑ їде вправо
- Nav-посилання: підкреслення виїжджає зліва направо
- Всі картки (проблема, послуги, прайс, відгуки, процес): підйом + збільшення тіні
- Форма: фокус з підсвіченою межею
- Мобільне меню: гамбургер → хрестик з анімацією

Усі анімації вимикаються, якщо у користувача `prefers-reduced-motion: reduce`.

## Адаптив — 5 breakpoint'ів

| Ширина          | Що змінюється                                            |
|-----------------|----------------------------------------------------------|
| **1280+ px**    | Повний десктоп: 5 послуг у ряд, 5 прайсів, 7 кроків, 4 відгуки |
| **1024–1279 px**| Прайс і послуги залишаються 5-у-ряд, але текст менший     |
| **768–1023 px** | Hero стає 1-колонка (картка знизу), послуги 3-у-ряд, прайс 3+2, відгуки 2×2, процес 4-у-ряд без лінії, trust 1-колонка |
| **640–767 px**  | Все на 2 колонки: послуги 2×2+1, прайс 2×2+1, процес 2×2 |
| **480–639 px**  | Мобільна: hero-картка залишається 2-кол, стрічки контенту |
| **< 480 px**    | Все в одну колонку, кнопки на всю ширину, зменшена типографіка |

**Мобільні особливості:**
- Меню: гамбургер відкриває вертикальний список з розділювачами
- Форма: `input font-size: 16px` — щоб iOS не зумив при фокусі
- `viewport-fit=cover` — коректно поводиться з notch на iPhone
- Мін. touch-target 44×44 px на всіх кнопках і посиланнях
- Header sticky, але вирівнюється відступом при smooth-scroll

## Запуск локально

```bash
# Просто відкрийте index.html у браузері
# або підніміть простий сервер (потрібно для деяких браузерів через CORS):
python3 -m http.server 8080
# → http://localhost:8080

# Node варіант
npx serve .
```

## Що треба зробити перед продакшеном

### 1. Форма → реальний backend
У `script.js` знайти коментар `Replace this block` і замінити demo-логіку.

**Telegram Bot (найпростіше):**
```js
const TG_TOKEN = 'YOUR_BOT_TOKEN';
const TG_CHAT  = 'YOUR_CHAT_ID';
const text = `📝 Нова заявка з Nova:\n\n` +
  `Ім'я: ${payload.name}\n` +
  `Телефон: ${payload.phone}\n` +
  `Telegram/Viber: ${payload.telegram || '—'}\n` +
  `Коментар: ${payload.comment || '—'}`;
await fetch(`https://api.telegram.org/bot${TG_TOKEN}/sendMessage`, {
  method: 'POST',
  headers: {'Content-Type':'application/json'},
  body: JSON.stringify({ chat_id: TG_CHAT, text })
});
```

Як створити бота: пишете **@BotFather** у Telegram → `/newbot` → отримуєте
токен. Свій chat_id можна дізнатись через **@userinfobot**.

### 2. Фото засновниці — self-host
Зараз фото підтягується з Framer CDN. Якщо CDN коли-небудь відключать —
збережіть фото як `assets/founder.jpeg` і в `index.html` замініть URL
у `<img src="...">` секції `#trust`.

### 3. Meta для соцмереж
У `<head>`:
- Замініть `og:url` на реальний домен
- Додайте `<meta property="og:image" content="URL до 1200×630 картинки">`

### 4. Аналітика та Pixel (опційно)
Додайте перед `</head>`:
- Google Analytics / GA4
- Meta Pixel — щоб відстежувати конверсії з реклами
- Google Tag Manager, якщо треба комплексно
